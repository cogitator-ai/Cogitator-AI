Initial release of Cogitator - self-hosted AI agent runtime for TypeScript.

Features:
- Agent runtime with tool execution and multi-LLM support
- DAG-based workflow engine with checkpoints
- Multi-agent swarm coordination
- MCP (Model Context Protocol) integration
- OpenAI-compatible API server
- Docker/WASM sandboxed code execution
- Distributed job queue with BullMQ
- Memory adapters (in-memory, PostgreSQL, Redis)
